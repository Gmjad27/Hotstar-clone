# 🎬 CineFlix – Movie & TV Streaming UI (React + Vite)

🔗 **Live Preview:** 👉 https://cine-flix.jadavgirish67.workers.dev/

CineFlix is a **modern, high-performance movie & TV streaming web application UI** built with **React + Vite**.  
It delivers a **fast, smooth, and visually rich OTT-like experience**, inspired by popular streaming platforms such as Netflix, Disney+, and Prime Video.

---

## ✨ Key Highlights

- 🚀 **Lightning Fast** – Powered by Vite with Hot Module Replacement (HMR)
- 🎨 **Cinematic UI** – Dark theme with OTT-style layouts
- 📱 **Fully Responsive** – Seamless experience on mobile, tablet & desktop
- 🧩 **Reusable Components** – Clean and scalable React architecture
- 🎞️ **Dynamic Content** – Movies, TV shows, studios & categories
- 🔍 **Search & Browse** – Easy navigation and filtering
- 🛠️ **Developer Friendly** – Organized project structure & linting

---

## 🖥️ Tech Stack

| Technology | Usage |
|-----------|------|
| ⚛️ **React** | UI development |
| ⚡ **Vite** | Fast build tool & dev server |
| 🎨 **CSS Modules** | Component-level styling |
| 🔁 **React Router** | Client-side routing |
| 📦 **ESLint** | Code quality & consistency |
